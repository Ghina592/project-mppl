# Sistem Informasi Sekolah

Sistem informasi sekolah berbasis laravel 11 dengan template dashboard

## Fitur

-   CRUD Jurusan
-   CRUD Mata Pelajaran
-   CRUD Guru
-   CRUD Kelas
-   CRUD User
-   CRUD Materi
-   CRUD Tugas & Jawaban
-   CRUD Jadwal Sekolah
-   CRUD Absensi
